AI Travel Chat Bot - PHP MySQL Project

How to run in XAMPP:
1. Copy folder ai_chatbot_project to C:/xampp/htdocs/
2. Start Apache and MySQL in XAMPP.
3. Open phpMyAdmin: http://localhost/phpmyadmin
4. Import database/ai_chatbot_db.sql
5. Open project: http://localhost/ai_chatbot_project/
6. Admin panel: http://localhost/ai_chatbot_project/admin.php

Database name: ai_chatbot_db
Default XAMPP username: root
Default password: empty

Learning feature:
Type in chatbot:
learn: beach tour = We offer beach tours in Mirissa, Hikkaduwa and Unawatuna.

Main AI concepts included:
- Natural Language Interface
- Inference Engine
- Knowledge Base
- Persistent Storage
- Simple learning by updating knowledge base
- Chat history storage
