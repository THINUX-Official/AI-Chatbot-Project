Technical Documentation Summary

Project Title: Ceylon Travel Assistant - AI Chat Bot

Problem Description:
Tourists often need quick information about Sri Lanka travel packages, destinations, booking steps and contact details. This chatbot provides a simple natural language interface where users can type questions and receive relevant answers.

Research:
The project is based on chatbot architecture concepts: Natural Language Interface, Inference Engine and Knowledge Base. It uses keyword matching and similarity scoring to map user questions to suitable answers. Dynamic package details are stored in MySQL, so updated travel data can be retrieved without changing source code.

System Architecture:
1. User Interface: index.php and JavaScript chat UI.
2. Inference Engine: functions.php processes user text and selects the best answer.
3. Knowledge Base: MySQL tables store facts, packages and learned answers.

P.E.A.S:
Performance: Correct answer, fast response, useful package details, learning ability.
Environment: Web browser, user questions, tourism database.
Actuators: Text replies shown in chat interface.
Sensors: User typed messages.

AI Traits:
1. Natural Language Processing: Cleans user text and compares keywords.
2. Searching: Searches knowledge base and package database.
3. Learning: User can teach the bot new answers using learn command.
4. Decision Making: Bot selects the best answer based on similarity score.

Algorithm:
Start → Receive user message → Clean text → Check learning command → Check package keywords → Search knowledge base → Calculate similarity score → Select best answer → Save chat history → Display reply → End

Testing Examples:
Input: hello
Expected Output: Greeting message

Input: show packages
Expected Output: List of travel packages from database

Input: contact number
Expected Output: WhatsApp and email details

Input: learn: visa = Please check visa requirements before traveling.
Expected Output: Bot learns new answer

Conclusion:
The project demonstrates an AI chatbot using a natural language interface, inference engine and database knowledge base. It is suitable as a simple AI coursework artifact and can be extended with advanced NLP or machine learning.
